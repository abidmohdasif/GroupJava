## Coin Game | Heads or Tails
### Game Concept:
•	Compete with an enemy by flipping a coin to get a specified combination.

•	Set a randomized combo, ex. Tails, Head, Head

•	Enemy and player flip their coins 3 times until they get the specified combo

•	First to get a combo 3 times wins.


One Player game, against AI (It’ll be simple, using only two possibilities, heads/tails)

•	Player goes first

•   During each round:

•	Print out the randomized flip combo

•	“Enter” to output results of coin flip

•	Put animation to indicate coin flips

•	“Enter” to output enemy’s results of coin flip

•	Put animation to indicate coin flips

•	Check if any are equal to randomized combo
